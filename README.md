# Student Manager (Flask + SQLite)

A simple CRUD web application built using Flask and SQLite.

## Features
- Add student
- View students
- Edit student
- Delete student

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS

## How to Run
1. Clone the repository
2. Install Flask
3. Run `python app.py`
4. Open http://127.0.0.1:5000/

## Project Structure
- app.py
- templates/
- static/
